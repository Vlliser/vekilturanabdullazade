Admin Panel haqqında

Bu admin panel vasitəsilə saytın bütün dinamik kontentini idarə edə bilərsiniz:

📋 Profil məlumatları — Ad, telefon, ünvan, sosial şəbəkə linkləri
💬 Rəylər — Müvəkkil rəylərini əlavə et, redaktə et, sil
🎓 Sertifikatlar — Sertifikat və diplomları idarə et
📺 Media çıxışları — TV, radio, mətbuat çıxışlarını əlavə et
💾 Məlumat yedəyi — Məlumatları JSON formatında export/import et

⚠️ Məlumatlar hazırda brauzer localStorage-da saxlanılır. GitHub Pages üçün uyğundur. Hostingerdə server backend əlavə etmək tövsiyə olunur.

Məlumat ixracı (Export)

Bütün kontent məlumatlarını JSON formatında yükləyin. Bu faylı ehtiyat nüsxə kimi saxlaya bilərsiniz.

Məlumat idxalı (Import)

Əvvəlcədən export edilmiş JSON faylını yükləyin. Bu, mövcud məlumatları əvəz edəcək.

Məlumatları sıfırla

Bütün kontent məlumatlarını ilkin (default) vəziyyətə qaytarın. Bu əməliyyat geri alına bilməz!
