# Vəkil Turan Abdullazadə — Rəsmi Sayt

## 📁 Fayl strukturu

```
turan-lawyer/
├── index.html          # Əsas səhifə (3 dilli: AZ/RU/EN)
├── css/
│   └── styles.css      # Bütün dizayn stilləri
├── js/
│   └── scripts.js      # Əsas JavaScript (i18n, animasiyalar, form)
├── admin/
│   ├── index.html      # Admin panel
│   ├── admin.css       # Admin panel stilləri
│   └── admin.js        # Admin panel məntiqi
└── images/
    ├── logo.png         # Vəkil loqosu
    └── turan-photo.jpg  # Vəkil fotousu
```

## 🚀 GitHub Pages-ə yükləmə

1. Yeni GitHub repo yaradın (məs. `vekil-turan`)
2. Bütün faylları repo-ya yükləyin
3. **Settings → Pages → Deploy from branch → main → / (root)**
4. Saytınız `https://USERNAME.github.io/vekil-turan` ünvanında aktiv olacaq

## 🌐 Hostingerdə yerləşdirmə

1. Hostingerdə File Manager açın
2. `public_html` qovluğuna bütün faylları yükləyin
3. Domain-i konfiqurasiya edin

## ⚙️ Admin Panel

**URL:** `/admin/index.html`

Admin paneldən aşağıdakıları idarə edə bilərsiniz:
- 👤 Profil məlumatları (ad, telefon, ünvan, sosial şəbəkə)
- 💬 Rəylər (əlavə et / redaktə et / sil)
- 🎓 Sertifikatlar (əlavə et / sil)
- 📺 Media çıxışları (əlavə et / sil)
- 💾 JSON Export / Import

> **Qeyd:** GitHub Pages-da admin panel localStorage ilə işləyir — məlumatlar brauzerinizdə saxlanılır. Hostinger-ə keçdikdən sonra PHP backend əlavə etmək tövsiyə olunur.
